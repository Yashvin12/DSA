## 🛠️ How to Compile & Run (C++)
To run any C++ code locally:
1. Open your terminal in the directory of the file (e.g., `Patterns`).
2. Compile the file:
   ```bash
   g++ -O2 -std=c++17 Pattern1.cpp -o Pattern1
   ```
3. Run the executable:
   - **Windows**: `.\Pattern1.exe`
   - **Mac/Linux**: `./Pattern1`
